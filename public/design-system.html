12 · Canva Export Guide
Cómo usar este Design System en Canva

Pasos para llevar tu sistema de diseño directamente al Brand Kit de Canva.

1
Exportar este archivo como PDF
Abre este archivo en Chrome → Archivo → Imprimir → Guardar como PDF. Asegúrate de activar "Gráficos de fondo". Luego en Canva: Importar → PDF para tener todas las páginas como referencia visual.
2
Configurar Brand Colors en Canva
Ve a Brand Kit → Brand Colors → añade los siguientes hexadecimales. Canva los guardará y aparecerán en el selector de colores de todos tus diseños.
#C4881E · Primary Light
#F59E0B · Primary Dark
#FFFBF2 · BG Light
#0C0802 · BG Dark
#1C1410 · FG Light
#F5F0E8 · FG Dark
#FEF3C7 · Secondary Light
#FBBF24 · Amber 200
#78716C · Muted Light
#ef4444 · Error
3
Configurar Brand Fonts en Canva
Ve a Brand Kit → Brand Fonts. Configura: Heading → DM Serif Display (búscalo por nombre, Canva lo tiene disponible). Body → Outfit (disponible en Canva). Mono / Labels → Geist Mono (si no aparece, usa Courier New como fallback para labels de código).
4
Crear elementos de botón como componentes
En un template de Canva, crea los 4 botones (Primary, Outline, Ghost, Link) con las medidas exactas de este sistema. Guárdalos como "Elementos de marca" para reutilizarlos. Usa esquinas redondeadas: 10px para botones, 18px para cards.
5
Escala tipográfica en Text Styles
En Brand Kit → Text Styles, configura: Heading (56px · DM Serif Display · Regular), Subheading (36px · DM Serif Display · Regular), Body (16px · Outfit · Regular). Para tamaños intermedios usa los valores de la sección 03 como referencia manual.
6
Usar este HTML como Cheat Sheet
Mantén esta página abierta mientras diseñas en Canva. Todos los valores exactos de hex, tamaños y espaciado están aquí para copy-paste directo. Puedes imprimir solo la sección de colores (sección 01) para tenerla física.
